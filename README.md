# E-Commerce Backend API

## Project Description

E-Commerce Backend API is a RESTful API built for managing an online store system.

The API provides complete management for categories, products, shopping cart, and orders with checkout functionality.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
- express-mongo-sanitize

---

# Features

## Categories API
- Create category
- Get all categories
- Get category by ID
- Update category
- Delete category

## Products API
- Create product
- Get all products
- Get product by ID
- Update product
- Delete product
- Filter products by:
  - Category
  - Price range
  - Search keyword

## Cart API
- View cart
- Add products to cart
- Update quantity
- Remove products
- Clear cart

## Orders API
- Checkout cart
- Create order
- Get all orders
- Get order details
- Update order status
