GitHub Repository:

About the Project:
This is my Level 4 First Term Web Development project.
It is a simple E-Commerce Backend API built using Node.js, Express.js, and MongoDB.

Features:
- Categories CRUD
- Products CRUD
- Shopping Cart
- Orders
- MongoDB Database
- Error Handling
- Seed Script

How to Run:
1. npm install
2. Create a .env file
3. npm run seed
4. npm run dev
